02 — STRUCTURAL ANALYSIS

2D Truss Analysis

Internal and external force analysis of members of a 2D truss structure.

PROJECT DATA

Type

FEA & Hand Calculation

Year

2024

Software

Ansys, Python, Excel

THE CHALLENGE

To accurately predict the failure points of a bridge truss design under varying load conditions. The goal was to verify manual calculations against FEA simulation results.

THE SOLUTION

A method of joints analysis was performed manually and verified using Python scripts. The structure was then modeled in Ansys to simulate stress distribution and deformation, confirming a safety factor of 2.5.

METHODOLOGY

Analytical vs. Numerical Approach

1. Hand Calculations

Using the Method of Joints, we isolated each node of the truss to solve for member forces. The structure was assumed to be statically determinate. We calculated reaction forces at the supports and propagated them through the web members.

2. FEA Simulation (Ansys)

The geometry was imported into Ansys Workbench. We applied a mesh size of 5mm using tetrahedral elements. Boundary conditions were set to fixed support at the left node and roller support at the right node.
